# MudSim接口函数参考手册

## 概述

MudSim是一款求解降雨诱发山洪泥石流的数值模拟软件。该软件可输入指定的降雨强度，分析不同降雨时间下的汇水、洪水演进、降雨及水流对松散物源的侵蚀、土颗粒的悬浮沉积等过程。

本手册主要供泥石流动力学分析软件MudSim的用户开展相应的数值计算使用。该手册提供了从网格导入、计算条件施加、核心计算及计算结果输出等的多类JavaScript接口函数，可方便用户开展泥石流运动堆积过程的分析。

# 公共接口

本章主要介绍泥石流模块的接口函数，函数列表见表1.1。

包括计算网格的导入、求解过程设置、全局量的设置及获取、降雨时程曲线的设置、颗粒级配曲线的设置、时程信息的监测、计算结果输出设置等。

表1.1 泥石流接口函数列表

| 序号 | 函数名         | 说明                         |
| ---- | -------------- | ---------------------------- |
| 1    | importGrid     | 导入计算网格                 |
| 2    | solve          | 核心求解，求解至设定时间退出 |
| 3    | setValue       | 设置全局变量                 |
| 4    | getValue       | 获取全局变量                 |
| 5    | setRainData    | 设置降雨时程曲线             |

### importGrid方法

#### 说明

导入GdemGrid或DEM格式的计算网格。

#### 格式定义

mflow.importGrid(*FileType*[, *FilePath*]);

#### 参数

*FileType*

文件类型，可以为ID号，也可以为字符串。

>（1）ID号输入为：1-从GdemGrid导入，2-从DEM导入。

>（2）字符串输入为：“GdemGrid”、“DEM”，大小写均可。

*FilePath*

>网格文件的路径及文件名。可以是完整路径，也可以是相对路径，或当前目录下的文件名。也可以为空，如为空，则会跳出对话框，让用户通过界面进行选取。
>
>目前，仅支持“GdemGrid”格式的网格。

#### 备注

（1）GdemGrid格式的网格文件如下图所示。第一行共4个数，分别是x方向点数Nx，y方向点数Ny，x方向间距Dx，y方向间距Dy；后续是Nx*Ny个点的坐标值（x,y,z）。

![](\pic\GdemGrid.jpg)

（2）如果*FilePath*中直接输入网格文件名，则从软件当前路径下读入网格文件（用户可通过在命令输入框中输入pwd()查看当前路径）。用户也可以在网格文件名的前面增加路径，此时的路径分割符需要用两个正斜杠“\\”或一个反斜杠“/”表示。

#### 范例

```js
//从GdemGrid导入文件
mflow.importGrid(1, “Grid.dat”);
mflow.importGrid(“GdemGrid”, “Grid.dat”);

//从DEM导入文件
mflow.importGrid(“DEM”, “Model.dat”);
```

### sovle方法

#### 说明

核心求解。

#### 格式定义

mflow.solve(*fTotalTime*);

#### 参数

*fTime*

> 浮点型，计算到多长时间，全量值（单位：s）。

#### 备注

#### 范例

```js
//计算到120s
mflow.sovle(120.0);
```

### setValue方法

#### 说明

设置全局参数数值。

#### 格式定义

mflow.setValue(*strName*, *fValue* [,*iflag*]);

#### 参数

*strName*

> 字符串型，设置变量的名称。

*fValue*

> 整型（I）或浮点数（F），设置变量的参数值。不同的参数名可能具有不同的参数类型。

*iflag*

> 整型，设置变量的分量ID号，如果为标量，可以不写，或写1。

#### 备注

每执行一次命令，对应的全局参数便进行一次重置。

可供设置的全局变量见附表1。

#### 范例

```js
//设置结果输出时间间隔
mflow.setValue(“Output_Interval”, 20.0);

//设置初始含水率
mflow.setValue(“InitWaterCont”, 0.03);
```

### getValue方法 

#### 说明

获得全局量，返回值可赋给JavScript变量。

#### 格式定义

mflow.getValue(*strName* [*,iflag*]);

#### 参数

*strName*

> 字符串型，获取变量的名称。

*iflag*

> 整型，获取变量的分量ID号，如果为标量，可以不写，或写1。

#### 备注

可供获取的全局变量见附表1。

#### 范例

```js
//获取材料黏聚力
var cohe = mflow.getValue(“cohesion”);

//获取重力加速度
var g = mflow.getValue(“gravity”,1);
```

### setRainData方法

#### 说明

设置降雨时程曲线。

#### 格式定义

mflow.setRainData(*afData\[n][2]*)

#### 参数

*afData\[n][2]*

> Array浮点型，包含n行（n≥2），2列，第一列为时间（s），第二列为降雨强度(m/s)。

#### 备注

降雨时程曲线中时间点数量n应大于等于2。

#### 范例

```js
//设置降雨时程曲线，时间从0s至1e6s，降雨强度为1e-3m/s
mflow.setRainData([0.0, 1e-3, 1e6, 1e-3]);
```



### setParData方法

#### 说明

设置颗粒级配曲线。

#### 格式定义

mflow.setParData(*afData\[n][2]*)

#### 参数

*afData\[n][2]*

> Array浮点型，包含n行（n≥1），2列，第一列为颗粒直径（mm），第二列为所占比例。

#### 备注

各粒径所占比例即第二列数据之和须等于1。

#### 范例

```js
//设置颗粒级配曲线
mflow.setParData([0.001, 0.4, 0.005, 0.4, 0.01, 0.2]);
```



### hist方法

#### 说明

监测某一测点的时程信息。

#### 格式定义

mflow.hist(*strName*, *fx*, *fy*)

#### 参数

*strName*

> 字符串型，监测内容。

*fx*, *fy*

> 浮点型，监测点的坐标。

#### 备注

监测内容可为：

”height”：水深；

”xvel”：x方向速度；

”yvel”：y方向速度；

”magvel”：合速度；

”c”：沉积物体积浓度；

”m”：单位面积沉积物总质量；

”dh”：沉积物厚度。

#### 范例

```js
//监测与（1024, 2230）点最近的节点的水深
mflow.hist("height", 1024, 2230);
```



### exportTextData方法

#### 说明

输出最终计算结果。

#### 格式定义

mflow.exportTextData()

#### 参数

#### 备注

在tmp文件夹下生成文件，记录最终时刻计算结果信息。

#### 范例

```js
//输出最终计算结果
mflow.exportTextData();
```



### drawHistPos方法

#### 说明

绘制监测点的空间位置，在模型视图中用红色圆点表示。

#### 格式定义

mflow.drawHistPos()

#### 参数

#### 备注

1. 当监测信息设置完毕，在计算之前，可通过该接口进行监测点位置的检查。

2. 监测点位置图形可通过平台提供的draw.clear()、draw.commit()函数进行清除。

#### 范例

```js
//绘制监测点的空间位置
mflow.drawHistPos();
```
