## CDEM动力学分析系统（CDyna）介绍

CDEM动力学分析系统（CDyna）是一款显式动力学分析软件，包含块体模块、颗粒模块、冲击波模块三个模块。该软件可以模拟多场耦合下固体材料的弹性、塑性、损伤、破裂过程，以及破碎后散体的运动、碰撞、流动、堆积过程，可广泛应用于岩土、爆炸冲击、采矿、隧道、油气、水利、地质、结构、机械等多个领域。

------

块体模块：基于多核CPU并行，实现了有限元与块体离散元的耦合，可精确刻画静、动载荷下固体的破裂演化过程及碰撞堆积过程，可模拟渗流、温度等多场耦合下的固体力学行为。

------

颗粒模块：基于多核CPU并行，包含颗粒离散元（DEM）、物质点法（MPM）、基于颗粒接触的无网格法（PCMM）、颗粒飞散算法（PFly）等多种粒子算法；通过与块体模块的耦合，可实现块体、颗粒耦合转化过程的模拟。

------

冲击波模块：基于多核CPU并行，可模拟固体炸药的起爆过程及可燃气体的燃爆过程，可模拟爆炸冲击波在空气中的传播过程；通过与块体或颗粒模块的耦合，可实现冲击波作用下固体材料渐进破坏过程的精确模拟。

------

### 块体模块介绍

------

块体模块以连续-非连续单元方法（CDEM）为计算核心，利用块体表征材料的连续介质特性，利用块体间的界面表征材料的非连续介质特性，通过块体边界及块体内部的断裂，实现材料渐进破坏过程的模拟。该模块不仅能模拟静、动力载荷下材料的弹性、塑性、损伤及破裂过程，还可以模拟破碎后散体的运动、碰撞、流动及堆积过程。该模块支持渗流、温度、应力、破裂等的交互耦合，可广泛应用于岩土、地灾、采矿、隧道、爆破、水电、能源等多个领域。

l 有限元与块体离散元的完美结合，包含18种内置单元本构，12种内置接触本构。

l 包含裂-孔隙耦合渗流模块、热传导模块、杆件模块、刚性面模块等，可以实现固体单元与多类其他单元的耦合计算。

l 以JavaScript脚本驱动为主，以GUI操作为辅，通过近500个接口函数，实现前后处理及计算过程的精确控制。

l 基于多核CPU进行并行加速，计算效率是计算核心数的0.7倍；

l 二次开发功能强大，可直接利用C++或JavaScript进行单元本构、接触本构的开发，核心求解流程的控制以及计算结果的定制化输出。

l 支持参数化创建复杂几何模型并剖分网格，支持6类主流软件（Patran、Ansys、FLAC3D、GiD、ABAQUS、Gmsh、Midas）的网格导入，同时支持二维、三维网格的力学计算。

l 数据统计及输出更加人性化，内置大量数据统计及输出策略，可对单元破碎块度、系统能量分布、系统平均场量等进行分别统计及输出。

l 内置大量时程监测变量，可对应力、应变、损伤度、破裂度等信息进行监测输出。

l 支持重启动分析，按需定时存储重启动文件，无惧计算机宕机。

### 颗粒模块介绍

------

该模块以颗粒离散元分析功能为主，包含众多广义粒子计算功能，如无网格分析功能（PCMM、MPM），颗粒阻力飞行分析功能等。在颗粒离散元子模块中，通过颗粒间连接键的损伤、断裂，可模拟材料从连续到非连续的渐进破坏过程；通过离散颗粒间的接触检测及接触力计算，可模拟散体材料的运动、碰撞及堆积过程。此外，借助无网格子模块可以准确模拟连续介质的大变形过程，借助PFLY子模块可以准确模拟颗粒群在空气及水中的减速运动过程。该模块可广泛应用于岩土、采矿、水电、隧道、冲击爆炸、地质灾害等多个领域。

#### 模块功能

| -------- | ------------ | ------------------------------------------------------------ |
| 基础模块 | 颗粒接触模型 | 开挖模型、Herz模型、线弹性模型、脆性断裂模型、理想弹塑性模型、应变软化模型、断裂能模型、自定义模型。 |
|          | 载荷边界     | 静力：速度、重力、节点力、面力；动力：速度、节点力时程曲线；阻尼：局部阻尼、粘性阻尼 |
|          |              |                                                              |
| 扩展包   | 刚性面模块   | 用于提供支撑边界，可施加刚体平动及转动条件。                 |
|          | 结构单元模块 | 桩、梁、锚杆、锚索、网模型，可模拟颗粒材料与结构单元的相互作用。 |
|          | PCMM模块     | 利用基于颗粒接触的无网格方法（PCMM），可用颗粒表征连续介质单元，并模拟材料的大变形过程。 |
|          | MPM模块      | 基于物质点算法，模拟材料的大变形问题。                       |
|          | PFLY模块     | 基于颗粒在空气中及在水中运动时的阻力系数变化特征，精确分析颗粒飞行轨迹及停积位置。 |

#### 模块特点

l 借助近200个接口函数，精确控制颗粒生成、本构设置、初边值条件施加、求解设置等环节。

l 通过多种方式快速生成指定分布的颗粒体系，支持从GiD中导入颗粒。

l 支持二维、三维的颗粒计算，颗粒离散元模块可同时考虑颗粒的平动及转动效应。

l 通过与块体模块的耦合，可以实现块体颗粒耦合转化过程的模拟。

l 二次开发功能强大，可借助JavaScript及C/C++进行自定义本构开发，并对求解流程进行控制；

l 强大的重启动分析功能，通过调入*.sav文件直接执行后续分析；

l 内置强大的参量时程监测功能，可监测颗粒、结构单元、刚性面的各类信息，同时可监测颗粒拉伸破裂数、剪切破裂数、拉伸损伤量、剪切损伤量等统计性指标。

### 冲击波模块介绍

------

该模块支持用户开展千万单元级工程问题的快速计算。该模块可模拟固体炸药的起爆过程及可燃气体的燃爆过程，可模拟爆炸冲击波在空气中的传播过程，可分析计算区域内压力、流速及温度的时空演化规律；此外，通过与块体模块及颗粒模块的耦合，可实现冲击波作用下固体材料毁伤过程的精确模拟。该模块采用有限体积法进行动力学方程的求解，采用投影算法处理时间步刚性问题，采用近似黎曼求解器Kurganov算法进行数值通量计算，采用MUSCL方法及二阶Runge-Kutta方法确保二阶时空精度。该模块可广泛应用于各类武器的冲击波毁伤效能评估及各类工程装备及工程结构的抗冲击性能评估。

#### 模块特点

l 采用正交网格技术，通过几何图元的布尔操作快速构建固体区域边界；

l 支持通过固体网格创建流体域中的固体边界；

l 以JavaScript脚本驱动为主，以GUI操作为辅，通过数十个接口函数，实现前后处理及计算过程的精确控制；

l 基于多核CPU进行并行加速，计算效率是计算核心数的0.7倍；

l 二次开发功能强大，可直接利用C++或JavaScript进行材料参数、计算流程及计算结果的个性化设置。
